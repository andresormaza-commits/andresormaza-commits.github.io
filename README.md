# Tríada interactiva — Resultados (Mapping UX)

Infografía controlada por la mano (MediaPipe). Archivo único: `index.html`.

## Publicar en GitHub Pages — 3 pasos
1. En github.com crea un repositorio nuevo (público) y sube el archivo `index.html`.
2. Entra a Settings → Pages. En "Branch" elige `main` y carpeta `/ (root)`. Guarda.
3. Espera ~1 minuto y abre la URL que aparece: `https://TUUSUARIO.github.io/NOMBRE-REPO/`

Esa URL queda fija y se abre siempre.

## Uso
- Permite el acceso a la cámara cuando el navegador lo pida.
- Mover la mano = navegar · juntar pulgar e índice = abrir tabla · Esc = cerrar.
- Funciona también con mouse.

## Notas
- Requiere internet (MediaPipe se carga desde un CDN).
- Recomendado abrir en Chrome.
