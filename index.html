Tríada interactiva de resultados

Controla la infografía con tu mano usando la cámara (MediaPipe). El cursor sigue la punta de tu índice. Pasa la mano sobre cada zona para resaltarla y cierra el puño para abrir su tabla. Con una tabla abierta, cierra el puño fuera del recuadro para volver al inicio.

Mover la mano = navegar  ·  Cerrar el puño = seleccionar / volver  ·  Esc = cerrar
¿Sin cámara? Pulsa Esc o haz clic aquí para usar el mouse.
